Ajouter un jeu dans l'application

Tout ce qui suit doit tre raliser dans l'application Angular.

Nous avons utiliser boostrap 4.0 et angular material pour le front du projet en grande partie vous pouvez donc allez piocher des styles directement depuis ces librairie pour le rendu front de votre jeu.

Dans le cas d'une introgation je vous conseille de regardez les autres componenent de jeu dans le dossier source (Angular).

Si votre jeu doit contenir des images : crer une liste d'Image qui contiendra le resultat de la mthode recup du component images.componenent.ts qui ira chercher les images dans la BDD.

Etape 1 : Crer le component pour le jeu

Rendez vous dans la racine du projet dans un shell et tapez la commandes quit suit 
$ ng generate c {NOM_DU_JEU}

Aprs avoir fait cette commande un nouveau dossier dans '/src/app/{NOM_DU_JEU}' devrait avoir t crer

  • Le fichier HTML permet de grer le contenu du component
  • Le fichier TS permet d'incluce le TypeScript du component
